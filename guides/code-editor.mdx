## Overview

Tesslate Studio uses the **Monaco Editor** (the same editor that powers VS Code) for a professional coding experience.

## Editor Layout

The code editor appears in the left panel of the project workspace:

```
┌─────────────────┬─────────────────┐
│  File Tree      │  Live Preview   │
│  ├── src/       │                 │
│  │   └── ...    │                 │
│  └── public/    │                 │
│                 │                 │
│  Code Editor    │                 │
│  [Your code]    │                 │
│                 │                 │
│  Chat Panel     │                 │
└─────────────────┴─────────────────┘
```

### Keyboard Shortcuts

<AccordionGroup>
  <Accordion icon="keyboard" title="File Operations">
    - **Ctrl/Cmd + S**: Manual save
    - **Ctrl/Cmd + P**: Quick file open
    - **Ctrl/Cmd + W**: Close current tab
    - **Ctrl/Cmd + Shift + F**: Search across files
  </Accordion>

  <Accordion icon="keyboard" title="Editing">
    - **Ctrl/Cmd + Z**: Undo
    - **Ctrl/Cmd + Shift + Z**: Redo
    - **Ctrl/Cmd + F**: Find in file
    - **Ctrl/Cmd + H**: Find and replace
    - **Ctrl/Cmd + D**: Select next occurrence
    - **Alt + Up/Down**: Move line up/down
  </Accordion>

  <Accordion icon="keyboard" title="Code">
    - **Ctrl/Cmd + /**: Toggle comment
    - **Ctrl/Cmd + ]**: Indent
    - **Ctrl/Cmd + [**: Outdent
    - **Alt + Shift + F**: Format document
    - **F2**: Rename symbol
  </Accordion>
</AccordionGroup>

### Error Detection

Real-time error checking:

- **Red Squiggles**: Syntax errors
- **Yellow Squiggles**: Warnings
- **Hover**: See error details
- **Quick Fix**: Click lightbulb for fixes

<Info>
  TypeScript errors appear in real-time as you type, helping catch issues before runtime.
</Info>

### Code Formatting

Keep code clean and consistent:

1. **Auto-Format**: Ctrl/Cmd + Shift + F
2. **Format on Save**: Enabled by default
3. **Prettier Integration**: Built-in formatting
4. **Customizable**: Adjust in settings

## Troubleshooting

<AccordionGroup>
  <Accordion icon="circle-exclamation" title="Editor Not Responding">
    **Problem:** Editor frozen or laggy

    **Solutions:**
    - Refresh the browser
    - Close unused tabs
    - Check browser console for errors
    - Disable browser extensions
  </Accordion>

  <Accordion icon="file-slash" title="File Won't Save">
    **Problem:** Changes not persisting

    **Solutions:**
    - Check file permissions
    - Ensure file isn't read-only
    - Try manual save (Ctrl/Cmd + S)
    - Refresh and try again
  </Accordion>

  <Accordion icon="lightbulb-slash" title="No IntelliSense">
    **Problem:** Auto-complete not working

    **Solutions:**
    - Wait for TypeScript to initialize
    - Check file has .ts or .tsx extension
    - Ensure package.json is valid
    - Restart development server
  </Accordion>
</AccordionGroup>
