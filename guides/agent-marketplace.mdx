## Installing Agents

<Steps>
  <Step title="Find Agent">
    Browse marketplace and find an agent you want
  </Step>

  <Step title="Review Details">
    Read description, features, and reviews
  </Step>

  <Step title="Click Install">
    Click **Install** or **Add to Library**
  </Step>

  <Step title="Confirm">
    Confirm installation (free) or payment (paid)
  </Step>

  <Step title="Go to Library">
    Agent appears in your Library → Agents
  </Step>

  <Step title="Enable">
    Toggle the agent on to use in projects
  </Step>

  <Step title="Start Using">
    Select it from agent dropdown in any project
  </Step>
</Steps>

<Info>
  Installing an agent doesn't automatically enable it. You control which agents appear in your selector.
</Info>

## Pricing Models

<Tabs>
  <Tab title="Free">
    **No cost agents**

    - No subscription fee
    - Full access to features
    - Often open source
    - Community supported
    - Pay only for AI API usage (your key)

    **Best for:** Most users, trying out agents
  </Tab>

  <Tab title="Subscription">
    **Monthly/yearly fee**

    - Premium features
    - Priority support
    - Regular updates
    - Exclusive capabilities
    - Still need your AI API keys

    **Best for:** Professional use, specialized needs
  </Tab>

  <Tab title="BYOK (Bring Your Own Key)">
    **Use your API keys**

    - Agent is free or subscription
    - You provide OpenAI/Anthropic API key
    - Pay AI providers directly
    - Full control over costs
    - Usage billed to your account

    **Best for:** Cost control, high usage
  </Tab>
</Tabs>

## Troubleshooting

<AccordionGroup>
  <Accordion icon="circle-xmark" title="Can't Install Agent">
    **Solutions:**
    - Check internet connection
    - Ensure you're logged in
    - Verify payment method (if paid)
    - Try refreshing page
    - Contact support
  </Accordion>

  <Accordion icon="ban" title="Agent Not Appearing">
    **Solutions:**
    - Check it's enabled in Library
    - Refresh project
    - Ensure agent supports your project type
    - Reinstall if needed
  </Accordion>

  <Accordion icon="triangle-exclamation" title="Agent Not Working Well">
    **Solutions:**
    - Check you're using it for intended purpose
    - Try more specific prompts
    - Compare with agent description
    - Leave feedback for creator
    - Try different agent
  </Accordion>
</AccordionGroup>
