## What is Tesslate Studio?

Tesslate Studio is an AI-powered development platform that helps you build complete web applications using natural language. Simply describe what you want to build, and our AI agents will generate production-ready React code with live preview.

## Key Features

<AccordionGroup>
  <Accordion icon="code" title="Full-Stack Code Generation">
    Generate complete React/TypeScript applications including components, state management, API integrations, and styling with Tailwind CSS. AI agents can read, write, and edit files, execute shell commands, and fetch web content.
  </Accordion>

  <Accordion icon="browser" title="Instant Live Preview">
    Every change you make or code the AI generates is instantly reflected in a live browser preview with hot module replacement. Each project runs in an isolated container environment.
  </Accordion>

  <Accordion icon="comments" title="Interactive Chat Interface">
    Chat with AI agents using natural language to describe features, request changes, or debug issues. Choose between Stream agents for fast iteration or Iterative agents for complex multi-step tasks.
  </Accordion>

  <Accordion icon="layer-group" title="Project Templates & Bases">
    Start from scratch, import from GitHub, or use pre-built bases with configured tech stacks and best practices. Support for multi-container monorepo projects.
  </Accordion>

  <Accordion icon="diagram-project" title="Architecture Visualization">
    Visualize your project architecture with React Flow-based diagrams showing container dependencies and connections in multi-container projects.
  </Accordion>

  <Accordion icon="github" title="GitHub Integration">
    Import repositories, edit code, and commit changes back to GitHub directly from the platform with full OAuth integration.
  </Accordion>

  <Accordion icon="cloud" title="Flexible Deployment">
    Deploy to Vercel, Netlify, or Cloudflare Pages with one click. Self-host with Docker for development or Kubernetes for production.
  </Accordion>

  <Accordion icon="credit-card" title="Billing & Marketplace">
    Purchase AI agents and project bases from the marketplace. Subscription plans with usage-based billing via Stripe.
  </Accordion>
</AccordionGroup>

## What You Can Build

Tesslate Studio is perfect for:

- **Landing Pages**: Create beautiful, responsive marketing sites
- **Web Applications**: Build full-featured SaaS applications with React + Vite
- **Full-Stack Apps**: Multi-container projects with frontend, backend, and database
- **Dashboards**: Generate data visualization and admin panels
- **E-commerce**: Create shopping experiences and product catalogs
- **Portfolios**: Build personal websites and project showcases
- **Prototypes**: Rapidly prototype ideas and validate concepts
