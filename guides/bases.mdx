## What are Bases?

Bases are pre-configured project templates that include:
- **Tech Stack**: Pre-installed libraries and frameworks
- **File Structure**: Organized folder layout
- **Sample Code**: Example implementations
- **Configuration**: Ready-to-use config files

## Finding Bases

Browse bases in the [Marketplace](/guides/marketplace):
1. Click **Marketplace** in the sidebar
2. Select **Bases** tab
3. Filter by category or search
4. View details and tech stack
5. Click **Install** to add to library

## Creating Projects from Bases

<Steps>
  <Step title="Install Base">
    Add the base to your library from the marketplace
  </Step>
  <Step title="Create Project">
    Click **Create New Project** from dashboard
  </Step>
  <Step title="Select Base">
    Choose **Base** as source and select from your library
  </Step>
  <Step title="Start Building">
    Project initializes with base structure ready to customize
  </Step>
</Steps>
