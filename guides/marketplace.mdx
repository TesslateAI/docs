## What is the Marketplace?

The Marketplace is your hub for discovering:
- **AI Agents**: Specialized coding assistants
- **Project Bases**: Pre-built templates
- **Tools**: Integrations and utilities
- **Integrations**: Third-party services

## Browsing the Marketplace

Access the marketplace:
1. Click **Marketplace** in the sidebar
2. Browse by category or search
3. Filter by type, pricing, or features
4. View detailed information
5. Install items to your library

## Item Types

### AI Agents
Specialized assistants for code generation:
- **Builder Agents**: General-purpose coding
- **Frontend Agents**: UI-focused development
- **Full-stack Agents**: End-to-end features
- **Data & AI Agents**: ML and data processing

### Bases
Pre-configured project templates:
- **Landing Pages**: Marketing sites
- **Dashboards**: Admin panels
- **E-commerce**: Shopping experiences
- **Applications**: Full-featured apps

### Tools & Integrations
Coming soon:
- **Design Tools**: Figma, Sketch import
- **APIs**: Third-party integrations
- **Deployment**: Hosting services

## Item Details

Each item shows:
- **Description**: What it does
- **Features**: Key capabilities
- **Tech Stack**: Technologies used
- **Pricing**: Cost model
- **Usage Count**: Popularity
- **Rating**: User feedback
- **Source Type**: Open or closed

## Open Source Items

Open source items can be:
- **Forked**: Create your own version
- **Modified**: Customize prompts and settings
- **Published**: Share back with community
- **Free**: No cost to use
